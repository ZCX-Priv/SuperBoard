## 项目概述
开发一款高性能的智能白板软件，支持Web和桌面端(Electron)，兼容鼠标精确指针和触摸屏粗略指针操作。

## 核心功能模块（基于设计图分析）

### 界面布局
1. **顶部灵动岛**: 显示当前时间 (X年X月X日 上午11时23分)
2. **白板内容区**: 黑绿色背景的画布区域
3. **底部工具栏**:
   - 左侧: 切换位置按钮
   - 中间: 画笔、板擦、选择、撤销、重做、收藏、更多
   - 缩放控制 (-100%+)
   - 返回底部按钮
   - AI助教按钮 (功能开启时显示)
   - 右侧: 工具、翻页&浏览、关闭白板

### 构建目标
- `npm run build:web` → Web版
- `npm run build:win` → Windows exe
- `npm run build:mac` → Mac app

### 输入支持
- 鼠标精确指针
- 触摸屏粗略指针
